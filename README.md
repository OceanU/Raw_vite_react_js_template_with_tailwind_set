React + Vite + Tailwind CSS Starter Template
A lightweight, high-performance starter template for React applications, pre-configured with Tailwind CSS, ESLint, and custom responsive design tools.

✨ Key Features
⚡ Vite 6 – Ultra-fast Hot Module Replacement (HMR).

🎨 Tailwind CSS v3 – Fully configured and ready to style.

📐 Extended Breakpoints – Built-in support for larger screens: 3xl (1920px) and 4xl (2560px).

🔍 Responsive Debugger – Custom function included to monitor real-time screen width in the console (showing both px and Tailwind's active breakpoint: sm, md, lg, etc.).

🧹 ESLint Config – Pre-set rules for maintaining clean and consistent code.

🛠️ Quick Start
Follow these steps to get your project up and running:

1. Use this Template
   Click the "Use this template" button at the top of this repository and "Create new repository" to create a new repo under your account.

2. Name your repository, fill details, set Public/Private and press "Create Repository"

3. Clone and Install
   git clone https://github.com/YOUR_USERNAME/YOUR_PROJECT_NAME.git
   cd YOUR_PROJECT_NAME
   npm install

4. Run Development Server
   npm run dev

5. Build for Production
   npm run build

🔧 Custom Configurations
Extended Breakpoints
We've updated the tailwind.config.js to include support for ultra-wide displays:

3xl: Optimized for 1920px screens.

4xl: Optimized for 2560px+ screens.

Console Screen Monitor
To make responsive design easier, this template logs the current window size and corresponding Tailwind breakpoint directly to your browser console. No more guessing when your layout will shift!
